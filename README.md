# EmotionSense-AI
AI-powered Emotion Detection System using Natural Language Processing (NLP), TF-IDF Vectorization, and Logistic Regression to classify text into emotions such as Joy, Sadness, Anger, Fear, Love, and Surprise. Includes text preprocessing, model evaluation, visualization, and real-time emotion prediction.
# EmotionSense-AI 🧠💬

## Overview

EmotionSense-AI is an NLP-based machine learning project that detects emotions from text. The system analyzes user input and predicts the underlying emotion such as:

* Joy 😊
* Sadness 😢
* Anger 😠
* Fear 😨
* Love ❤️
* Surprise 😲

The project demonstrates the complete NLP pipeline including text preprocessing, feature extraction, model training, evaluation, and prediction.
